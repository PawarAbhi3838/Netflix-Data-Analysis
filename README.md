# 🎬 Netflix Data Analysis

A comprehensive Exploratory Data Analysis (EDA) project on the Netflix Movies & TV Shows dataset using Python. This project uncovers trends, patterns, and insights about Netflix content through data cleaning, visualization, and statistical analysis.

---

## 📌 Project Overview

The objective of this project is to analyze the Netflix dataset and answer important business questions such as:

- How has Netflix's content grown over the years?
- Which countries produce the most Netflix content?
- What is the distribution of Movies vs TV Shows?
- Which ratings are most common?
- Which genres dominate the platform?
- How has content production changed over time?

The project demonstrates practical data analysis skills using Python libraries including Pandas, NumPy, Matplotlib, and Seaborn.

---

## 📂 Dataset

Dataset: **Netflix Movies and TV Shows**

The dataset contains information such as:

- Show ID
- Title
- Type (Movie / TV Show)
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre
- Description

## 📊 Data Analysis Workflow

### 1. Data Collection

- Imported Netflix dataset
- Loaded data using Pandas

### 2. Data Cleaning

- Checked missing values
- Removed duplicates
- Converted date columns
- Handled null values
- Corrected data types

### 3. Exploratory Data Analysis (EDA)

Performed analysis on:

- Movies vs TV Shows
- Content added over the years
- Country-wise content distribution
- Top directors
- Top genres
- Rating distribution
- Release year trends
- Duration analysis

### 4. Data Visualization

Created visualizations using:

- Bar Charts
- Count Plots
- Pie Charts
- Histograms
- Heatmaps
- Line Charts

---

## 📈 Key Insights

- Movies make up a larger portion of Netflix's catalog than TV Shows.
- Netflix experienced rapid content growth after 2015.
- The United States contributes the highest amount of content.
- TV-MA is the most common content rating.
- Drama and International Movies are among the most popular genres.
- Most movies have a runtime between 80 and 120 minutes.
